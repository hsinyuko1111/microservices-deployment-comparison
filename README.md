# Microservice Extravaganza

E-commerce microservices application with load balancing and message queue integration.

## Architecture

- **Product Service**: Manages product catalog
- **Shopping Cart Service**: Handles cart operations and checkout
- **Credit Card Authorizer**: Mocks payment processing
- **Warehouse Consumer**: Processes orders via RabbitMQ

## Services

- Product Service: Port 8080
- Shopping Cart Service: Port 8081
- Credit Card Authorizer: Port 8082
- RabbitMQ: Port 5672 (AMQP), 15672 (Management UI)
